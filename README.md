TurnosApp es una aplicación web desarrollada para la gestión de turnos médicos.

La plataforma permite que los pacientes se registren, completen su perfil con datos personales y soliciten turnos con profesionales de la salud. Cada paciente puede visualizar y gestionar únicamente su propia información y sus turnos.

El sistema cuenta con:
- Registro y autenticación de usuarios
- Perfil de paciente con datos personales
- Carga y edición de foto de perfil
- Gestión de turnos médicos
- Visualización de profesionales y especialidades
- Panel de administración para gestión completa del sistema

La aplicación está pensada para facilitar la organización de turnos y mejorar la experiencia tanto de pacientes como de administradores.
